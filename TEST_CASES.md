# 25 тест-кейсов Agent 3 PM Tracker

Веб: http://82.25.60.124:8083
Бот: Telegram

## Регистрация и вход

**TC-1. Регистрация нового пользователя**
1. Открой бота, нажми /start
2. Нажми «Зарегистрироваться»
3. Введи имя и фамилию
4. Выбери должность (должны быть 3 группы: 1 уровень, 2 уровень, 3 уровень)
5. Бот подтверждает регистрацию и показывает меню с 5 кнопками
- Ожидание: кнопки Мои задачи, Просрочки, Задать задачу, Спросить по задачам, Инструкции

**TC-2. Вход в веб по ссылке из бота**
1. В боте нажми «Мои задачи» или /start
2. Скопируй ссылку http://82.25.60.124:8083/enter/...
3. Открой в браузере
- Ожидание: автологин, попадаешь на канбан

**TC-3. Кнопка «Инструкции»**
1. Нажми кнопку «Инструкции» в боте
- Ожидание: подробное описание всех функций

## Создание задач через бота

**TC-4. Создание задачи текстом**
1. Нажми «Задать задачу»
2. Напиши: «Сделать лендинг для Амира срочно P0»
3. Агент должен спросить: «На какую доску?» с кнопками
4. Выбери доску
5. Агент спросит: «На какой этап?» с кнопками
6. Выбери этап
7. Агент покажет карточку и спросит про файлы
8. Нажми «Нет, создать»
- Ожидание: задача создана, ссылка в ответе, задача видна на канбане в веб

**TC-5. Создание задачи голосовым**
1. Нажми «Задать задачу»
2. Отправь голосовое: «Поставь задачу Амиру сделать дашборд, приоритет P1, дедлайн через неделю»
3. Бот: «Распознаю...» → «Распознано: ...»
4. Далее тот же флоу — доска, этап, файлы
- Ожидание: задача создана корректно из голоса

**TC-6. Создание задачи с файлами**
1. Нажми «Задать задачу»
2. Опиши задачу текстом
3. Пройди выбор доски и этапа
4. На вопрос «Прикрепить файлы?» нажми «Да»
5. Отправь фото или документ
6. Нажми «Готово»
- Ожидание: задача создана, файл в комментариях на странице задачи в веб

**TC-7. Агент НЕ создаёт задачу без доски и этапа**
1. Нажми «Задать задачу»
2. Напиши: «Задача — проверить отчёт»
3. Агент должен спросить доску (НЕ создать задачу автоматически)
- Ожидание: clarify с вопросом про доску, НЕ create_task

## Просмотр задач в боте

**TC-8. Кнопка «Мои задачи»**
1. Назначь себе задачу через веб
2. В боте нажми «Мои задачи»
- Ожидание: список задач со ссылками. Задача из п.1 в списке

**TC-9. Кнопка «Просрочки»**
1. Создай задачу с дедлайном вчера, назначь на себя
2. Нажми «Просрочки»
- Ожидание: задача в разделе «Просрочено»

**TC-10. Мусорный текст вне режима**
1. Просто напиши боту «привет как дела» (без нажатия кнопок)
- Ожидание: «Выбери действие кнопкой меню», НЕ отправка в GPT

## Управление задачами через бота (Спросить по задачам)

**TC-11. Спросить о задачах**
1. Нажми «Спросить по задачам»
2. Напиши: «Какие задачи у Амира?»
- Ожидание: список задач Амира со ссылками

**TC-12. Переставить задачу на следующий этап**
1. Нажми «Спросить по задачам»
2. Напиши: «Переставь задачу [название] на следующий этап»
- Ожидание: «Обновлено: ...» + ссылка. В веб статус изменился

**TC-13. Перенести задачу между досками**
1. Создай задачу на доске MarketAI Dev
2. Нажми «Спросить по задачам»
3. Напиши: «Перенеси задачу [название] на доску MarketAI Marketing»
- Ожидание: задача переехала на другую доску

**TC-14. Отметить задачу выполненной**
1. Нажми «Спросить по задачам»
2. Напиши: «Отметь задачу [название] выполненной»
- Ожидание: статус = done

**TC-15. Удалить задачу через бота**
1. Нажми «Спросить по задачам»
2. Напиши: «Удали задачу [название]»
- Ожидание: «Задача удалена: ...», исчезла из канбана

**TC-16. Напоминание через N минут**
1. Нажми «Спросить по задачам»
2. Напиши: «Напомни мне через 1 минуту проверить задачу»
- Ожидание: «Напомню через 1 мин.» → через минуту приходит «Напоминание: проверить задачу»

## Веб-интерфейс

**TC-17. Канбан — переключение досок**
1. Открой http://82.25.60.124:8083/board
2. Нажми на «MarketAI Dev» → видны задачи Dev
3. Нажми на «MarketAI Marketing» → видны задачи Marketing
- Ожидание: разные задачи на разных досках

**TC-18. Создание задачи через веб**
1. На канбане нажми «+ Добавить задачу» под любой колонкой
2. Заполни название, исполнителя, приоритет
3. Нажми «Создать»
- Ожидание: задача появилась в нужной колонке

**TC-19. Редактирование задачи**
1. Кликни на задачу → откроется полная страница
2. Измени название, приоритет, дедлайн
3. Нажми «Сохранить»
- Ожидание: вернуло на канбан, изменения сохранены

**TC-20. Кнопка «Выполнено» на странице задачи**
1. Открой задачу
2. Нажми зелёную кнопку «Выполнено»
- Ожидание: задача в колонке Done, вернуло на канбан

**TC-21. Удаление задачи через веб**
1. Открой задачу
2. Нажми красную кнопку «Удалить», подтверди
- Ожидание: задача исчезла с канбана

**TC-22. Комментарий с фото**
1. Открой задачу
2. Внизу напиши комментарий и прикрепи фото
3. Нажми «Отправить»
- Ожидание: комментарий появился, фото отображается inline

**TC-23. Копирование ссылки на задачу**
1. Открой задачу
2. Нажми «Копировать ссылку»
3. Открой ссылку в другом браузере/инкогнито
- Ожидание: задача открывается по ссылке

2. Залогинься как пользователь с должностью МОП → открой /employees
- Ожидание: редирект на /board, страница недоступна
